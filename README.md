Данный ПЕТ проект представляет реализацию веб-приложения фотосалона.
Приложение состоит из трёх основых вкладок:
1. Главная - основная информация о фотосалоне.
2. Услуги - перечень услуг, которые оказывает фотосалон. При нажатии на определенную услугу, можно посмотреть её подробное описание.
3. Контакты - основная контактная информация.
В корневой дирректории проекта лежит база данных "PhotoCompany".

При написании данного приложения был использован следующий стек:
1. c# (.Net 10)
2. Тип приложения: ASP .Net Core MVC
3. Visual Studio 2026
4. MS SQL Srver 2022
5. Entety Framework Core + Migrations + Identity
6. Serilog
7. HTML5
8. JavaScript
9. SCSS

При написании данного  веб приложения применялись следующие паттерны и подходы:
1. Dependency Injection (DI, внедрение зависимостей)
2. Асинхронный код (async/await)
3. Data Transfer Object (DTO)
4. Bundling
5. Minification
6. Логирование данных (Logs)
7. Domain Driver Design (DDD-подход)
8. Подход Code First
9. Аутентификация и авторизация (Authentication & authorization)
10. Миграции (Migrations)
11. Repository Pattern (Репозитории)
